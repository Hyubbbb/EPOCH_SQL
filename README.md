---



### 제출 양식
- **제출 경로**:
  1. **Essential**: `EPOCH_SQL/주제(SELECT 등)/Problem_N`
  2. **Optional**: `EPOCH_SQL/주제(SELECT 등)/Optional_N_n`
- **파일명**:
  1. **Essential**: `problem_N_solution_이름.sql` (_이름은 자유롭게 입력_ )
  2. **Optional**: `optional_N_n_solution_이름.sql` (_이름은 자유롭게 입력_ )

- **제출 기한**: 정기 세션 시작 전까지
---



## 문항 주제

1. `SELECT`
2. `SUM`, `MAX`, `MIN`
3. `GROUP BY`
4. `IS NULL`
5. `JOIN`
6. `STRING`, `DATE`
